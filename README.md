# 🚢 Batalha Naval

Projeto desenvolvido em Python que implementa o clássico jogo **Batalha Naval** em modo texto.
O sistema permite disputar partidas, registrar o desempenho dos vencedores e visualizar um ranking ordenado dos jogadores.

## Funcionalidades

- 🎮 Jogar partidas de Batalha Naval
- 🏆 Registro das estatísticas dos jogadores vencedores
- 📊 Exibir o ranking ordenado por desempenho
- 📋 Visualização das estatísticas de todos os vencedores
- ✔️ Tratamento de exceções para entradas inválidas

## Como executar o projeto

1. Clone o repositório

```bash
git clone <url-do-repositorio>
```

2. Acesse a pasta do projeto

Após clonar este repositório, abra seu terminal (Prompt de Comando no Windows), acesse a pasta do projeto e execute, com o seguinte comando:

```bash
python FrontEnd.py
```

> É necessário ter o Python 3 instalado
